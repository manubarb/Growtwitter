# GrowTwitter
Este projeto foi desenvolvido como **trabalho de conclusão do curso de Frontend com ReactJS** da GrowDev. O objetivo era criar um clone simplificado do Twitter desenvolvido com React e Redux Toolkit, focado em gerenciamento de estado complexo.

## Funcionalidades
- 🔐 **Autenticação** - Login de usuários
- 💾 **Persistência de dados** - Mantém a sessão do usuário
- 📝 **Tweets** 
  - Listagem
  - Criação de tweets
  - Exclusão de tweets
- 👤 **Perfil de usuário** - Visualização de informações do perfil
- 🌓 **Modo Light/Dark** - Alternância entre temas

## Tecnologias Utilizadas
- React.js
- Typescript
- Vite
- React Hooks
- Componentização
- Estilização de componentes
- Consumo de APIs
- Material UI
- Redux / Redux Toolkit

## Como rodar o projeto

### Pré-requisitos
Antes de começar, você vai precisar ter instalado em sua máquina:
- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)
  
### Clone o repositório:
```bash
git clone https://github.com/manubarb/Growtwitter.git
```
### Instale as dependências:
```bash
cd Growtwitter
npm install
```
### Configure as variáveis de ambiente:
```bash
#Crie um arquivo .env na raiz do projeto com a URL da API
VITE_API_BASE_URL=https://growtwitter-api-dny6.onrender.com/
```
### Inicie o servidor:
```bash
npm run dev
```
